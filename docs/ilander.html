About

Nebula Navigator is a co-op 2D platformer-shooter where players navigate deadly space corridors, dodging obstacles and blasting through hazards. Designed in Unity with C#, it blends precision movement, shooting mechanics, and power-ups for a fun yet challenging experience.

Project Details

  • My Role: Gameplay Programmer
  • Team Size: 1
  • Engine: Unity (C#)
  • Development Time: 1 week

Nebula Navigator

Introduction

Nebula Navigator is my first game development assignment, created to solidify my understanding of core game mechanics, movement, shooting, and level structuring in Unity. The game is a 2D cooperative space platformer, where two players must fly, dodge, and shoot their way through tight corridors filled with obstacles.

Gameplay and Features

  • Co-op gameplay: Two players work together to clear obstacles and progress.
  • Flight-based movement: Navigate tight spaces using WASD/Arrow keys.
  • Shooting mechanic: Destroy obstacles with precise projectile shots using Space/K key.
  • Power-ups:
    • Speed Boost: Temporarily increases movement speed.
    • Slow Boost: Temporarily slows movement for better control.
    • Multi-Shot: Enables multiple projectiles for a short duration.
  • Physics-based interactions: Players bounce off each other, adding an unpredictable challenge, especially in tight spaces.
  • Game loop: Players progress through levels until both are eliminated, leading to a restart.

Development Process & Lessons Learned

  • Physics & Movement - Learned how to use rigidbodies, velocity control, and smooth movement mechanics.
  • Game Flow & UI - Implemented menus, pause functionality, and scene transitions.
  • Debugging & Iteration - Overcame issues with collision handling, basic shooting, and movement behavior.

While relatively simple, this game established my foundation in game development, teaching me how to structure mechanics, balance challenges, and optimize player experience on a basic level.
